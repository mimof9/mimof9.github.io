命令行语法格式

1. 描述命令行参数，一般参数的格式如下：

命令 <必选参数1|必选参数2> [-option {必选参数1|必选参数2}] [可选参数] {(默认参数)|参数|参数}

2. 符号含义

  • <>： 必选参数
  • {}： 必须在{}内给定的选择中选一个
  • []： 可选参数
  • ()： 必选参数默认值，只用在{}种
  • |： 或
  • …： 任意多个参数
